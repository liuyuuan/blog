## 2. 评分框架（总分 100 分）

### 维度1：护城河 (Economic Moat) — 20 分

**巴菲特原话：** "在商业中，我寻找被不可攻破的护城河保护的经济城堡。"

**评估要素：**
- 品牌优势：消费者是否愿意为品牌支付溢价
- 网络效应：用户越多，产品/服务越有价值
- 转换成本：客户更换供应商的成本有多高
- 规模效应/成本优势：是否有显著的规模经济

**量化指标：**

| 指标 | 计算公式 | 评分规则 |
|------|----------|----------|
| 毛利率稳定性 | std(近5年毛利率) | < 3%: 8分, < 5%: 6分, < 8%: 4分, ≥ 8%: 2分 |
| 毛利率水平 | mean(近5年毛利率) | > 60%: 6分, > 40%: 5分, > 30%: 4分, > 20%: 3分, ≤ 20%: 1分 |
| ROE持续性 | 近5年ROE均 > 15%的年数 | 5年全达标: 6分, 4年: 5分, 3年: 4分, 2年: 3分, ≤ 1年: 1分 |

**满分条件：** 毛利率 > 60% 且波动 < 3%，ROE 连续5年 > 15%

### 维度2：安全边际 (Margin of Safety) — 20 分

**格雷厄姆原话：** "安全边际的概念是整个投资方法的基石。"

**DCF 估值模型参数：**

```
内在价值 = Σ(FCFt / (1 + WACC)^t) + 终值 / (1 + WACC)^n

其中：
- FCF (自由现金流) = 经营现金流 - 资本支出
- WACC (加权平均资本成本):
  - 无风险利率: 10年期美国国债收益率（默认 4.5%）
  - 市场风险溢价: 5.5%
  - Beta: 使用 yfinance 提供的 beta 值
  - 债务成本: 利息支出 / 总债务
  - 税率: 有效税率或 21%（美国）
- 增长率假设:
  - 前5年: min(历史FCF增长率, 15%)  # 乐观但有上限
  - 6-10年: 逐步衰减至终端增长率
  - 终端增长率: 3%（长期GDP增长+通胀）
- 终值 = FCF_n × (1 + g) / (WACC - g)
```

**Owner Earnings（巴菲特偏好的指标）：**
```
Owner Earnings = 净利润 + 折旧摊销 - 维护性资本支出 - 营运资金变动
其中维护性资本支出 ≈ 资本支出 × 0.7（粗略估计）
```

**评分规则：**

| 安全边际 | 得分 |
|----------|------|
| ≥ 50% | 20 分 |
| ≥ 40% | 16 分 |
| ≥ 30% | 12 分 |
| ≥ 20% | 8 分 |
| ≥ 10% | 4 分 |
| < 10% | 0 分 |

### 维度4：ROE与资本效率 — 15 分

**巴菲特原话：** "我们使用的首要测试是ROE——已运用的净资产的回报率。"

| 指标 | 计算公式 | 评分规则 |
|------|----------|----------|
| ROE水平 | 5年平均ROE | > 25%: 8分, > 20%: 7分, > 15%: 6分, > 10%: 4分, ≤ 10%: 2分 |
| ROIC vs WACC | ROIC - WACC | > 10%: 7分, > 5%: 6分, > 2%: 5分, > 0%: 3分, ≤ 0%: 0分 |

```
ROIC = NOPAT / 投入资本
NOPAT = EBIT × (1 - 税率)
投入资本 = 总资产 - 非经营性现金 - 无息流动负债
```

## 3. 筛选流程

### 第一轮：量化初筛（~5000 → ~100）

从主要指数成分股出发：
- S&P 500
- NASDAQ 100
- 恒生指数成分股
- 沪深300
- 日经225 / TOPIX 核心30
- FTSE 100 / Euro Stoxx 50

**硬性门槛：**
1. 市值 > 10亿美元
2. P/E > 0 且 < 行业中位数的 1.5 倍（排除亏损和极端高估）
3. P/B < 3
4. ROE > 15%（5年均值）
5. 连续盈利 ≥ 5年
6. 债务/EBITDA < 4

### 第二轮：深度分析（~100 → ~20）

1. DCF 估值：计算内在价值，要求安全边际 ≥ 20%
2. Owner Earnings：验证盈利质量
3. 护城河初步评估：毛利率趋势、ROE持续性

### 第三轮：综合排名（→ Top 5）

按8维度加权总分排序，取前5名。
